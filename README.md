<h1 align="center">
    Tokyo Culinária – Experiência Gastronômica Premium
</h1>

Website de um restaurante premium de culinária japonesa desenvolvido com foco em design elegante, navegação intuitiva e apresentação imersiva dos destaques do cardápio, incluindo seção de experiência, galeria, depoimentos e integração com WhatsApp para reservas.

---

## 📄 Descrição

Este projeto consiste em um website para um restaurante premium de culinária japonesa (Tokyo Culinária) desenvolvido para praticar conceitos fundamentais de desenvolvimento front-end, como:

- Estrutura de componentes reutilizáveis e escaláveis
- Estilização avançada com TailwindCSS e design tokens
- Seções de conteúdo estático com layout responsivo
- Integração com WhatsApp para reservas
- Animações suaves e efeitos visuais
- Organização semântica de componentes por recurso

O site apresenta uma experiência imersiva com seções de apresentação da marca, cardápio em destaque, galeria visual, depoimentos de clientes e navegação responsiva para mobile e desktop.

---

## ⚙️ Funcionalidades

- Página inicial (Hero) com apresentação premium da marca
- Seção de experiência com informações sobre a filosofia do restaurante
- Menu em destaque com pratos sazonais e descrições detalhadas
- Galeria visual de imagens do ambiente e pratos
- Seção de depoimentos de clientes
- Header com navegação responsiva
- Menu mobile para dispositivos menores
- Botão de WhatsApp para contato e reservas
- Footer com informações institucionais
- Layout totalmente responsivo (mobile e desktop)

## ▶️ Como rodar o projeto localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

```bash
# Clone o repositório
git clone https://github.com/MadeiraVitor/tokyo-culinaria.git

# Acesse a pasta do projeto
cd tokyo-culinaria

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O projeto estará disponível em:
http://localhost:5173
